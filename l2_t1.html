1. Дан код: var a = 1, b = 1, c, d;

c = ++a; alert(c); // 2
c = 2 потому что в начале идет увеличение переменной а на единицу, а потом присваивание значения к переменной с

d = b++; alert(d); // 1
к переменной d в начале присваивается значение b и потом идет увеличение переменной b

c = (2+ ++a); alert(c); // 5
в этом месте мы присваиваем новое значение с (2 + 3) 3 получилось потому что в начале увеличили переменную а на 1

d = (2+ b++); alert(d); // 4
тут мы тоже присваиваем новое значение переменной, но в отличии от предыдушего увеличение переменой б происходит после сложения с числом 2

alert(a); // 3
alert(b); // 3
из за предыдуших команд (++a и b++) изначальные переменные увеличились на 2 единицы Почему код даёт именно такие результаты?
